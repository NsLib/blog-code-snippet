## 配置

这里不是一个开箱即用的完整方案, 而是需要你自行整合进自己现有的 `zsh` 配置中:

* `add-index` 文件放到自己喜欢的 `bin` 目录, 例如:
    * `~/.local/bin/add-index`
    * `/usr/local/bin/add-index`
* `theme.yml` 文件放到 `~/.config/eza/theme.yml` (注意: `eza` 适配了这个主题的配色, 想用其他的要自行适配)
* `zshrc` 则整合进自己的 `~/.zshrc`
